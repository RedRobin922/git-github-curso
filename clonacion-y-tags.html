Clonacion y Tags de Git

Para agregar una etiqueta (tag) en git, se introduce el comando 'git tag', seguido de la fecha de la etiqueta, seguidamente se coloca -m, luego la descripcion entre comillas ("").
Por ejemplo: git tag 22-06-2022 -m "version 01 del proyecto"
Una etiqueta nos ayuda a identificar cuando se "lanza" la primera version de nuestro proyecto. Es decir, a parti de este punto, todos los cambios en el codigo del proyecto, pasan a ser parte de una nueva version del proyecto mismo.
Luego, para cargar las tags en el repositorio remoto, basta con introducir el comando 'git push --tags'.
